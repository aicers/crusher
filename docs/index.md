# Crusher

Crusher generates statistics from raw events. It connects to
[Giganto](https://github.com/aicers/giganto) to subscribe to network
event streams, computes time-series data according to sampling policies
managed by the [REview](https://github.com/aicers/review) manager
server, and sends the resulting statistics back to Giganto for storage
and analysis.

For source code and issue tracking, visit the
[GitHub repository](https://github.com/aicers/crusher).
